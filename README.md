## 它到底能干什么

### 剪贴板从"一格"变成"无限格"

系统自带的剪贴板只有一个格，复制新内容就把旧的顶掉了。PasteNow 相当于把这个格子扩充成无限个，文本、图片、链接、代码，复制过的全部都存着。

用快捷键 `CMD+Shift+V` 随时唤出列表，直接选，直接粘。全程不用离开键盘，效率差距立竿见影。

### 智能列表：乱糟糟的复制历史变得有条理

这个功能设计得挺有意思，灵感来自邮箱里的"智能邮箱"概念。你可以按内容类型（文字、图片、链接、颜色代码）、来源应用（Safari、Chrome、Sketch）、创建时间等规则，自动把剪贴板内容分好类。

对设计师来说有个特别贴心的小细节：复制 HEX 颜色代码的时候，PasteNow 会自动识别并显示对应色块，比如 `#FF5733` 旁边直接出现那个颜色的预览——不用再去色卡里一个个核对了。

### iCloud 跨设备同步

这是很多人购买的核心理由。Mac 上复制的内容，iPhone 上可以直接粘贴。数据走 iCloud 同步，不经过第三方服务器，隐私有保障。

比系统自带的"通用剪贴板"更稳，同步速度也更快。

### 隐私保护：敏感 App 可以排除在外

担心密码、银行账号被记录进去？可以把 1Password、支付宝这类敏感 App 加进排除列表，PasteNow 会自动忽略从这些 App 复制的内容。这个功能对安全要求高的用户来说挺关键的。
